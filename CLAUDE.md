## Design system

When building or modifying UI, you MUST read the following files before writing any code:
- node_modules/@syntax-design/syntax-ui/CLAUDE.md - entry point and file index
- node_modules/@syntax-design/syntax-ui/.ai-context/OVERVIEW.md — hard constraints, anti-patterns, compliance checklist (read this for every UI task, no exceptions)

Then read any additional .ai-context/ files relevant to the task.
The file index in OVERVIEW.md lists which files apply to which task types. Follow all rules and the compliance checklist in OVERVIEW.md before considering any UI task complete.
